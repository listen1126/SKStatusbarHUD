# SKStatusbarHUD
顶部弹窗(状态栏指示器 )

##显示成功信息
...objc
[SKStatusbarHUD showSuccess:@"加载成功"];
...

##显示失败信息
...objc
[SKStatusbarHUD showError:@"加载失败"];
...

##显示加载信息
...objc
[SKStatusbarHUD showLoding:@"加载ing"];
...

##显示文字信息
...objc
[SKStatusbarHUD showMessage:@"我就是测试呢~~看啥看"];
...

##显示隐藏信息
...objc
[SKStatusbarHUD dismiss];
...
